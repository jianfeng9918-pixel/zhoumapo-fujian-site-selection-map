# 周麻婆福建选址决策地图

内部选址前置筛选工具。页面主线为：福建省总览 → 城市 → 区县 → 镇街详情，用于判断今年开店优先看哪里、为什么看、适合什么店型，以及下一步查租金、补平台数据还是实地观察。

## 当前版本

- 版本：1.6 数据分包提速
- 地图：Leaflet + OpenStreetMap 真实底图，本地 GeoJSON 作为备用
- 数据：福建 9 城、84 县区、1121 镇街、158 家周麻婆门店、460 条五大竞品、3526 个公开/示意证据点
- 加载：`preview-data-core.json` 负责首页，`data/city-packs/` 按城市加载下钻数据，避免首次打开下载单个大型 JSON
- 数据源：官方/准官方统计、公开商业资料、OSM/公开 POI、用户/平台样本和模型推断分层使用

## 本地预览

```bash
python -m http.server 8788
```

然后打开：

```text
http://127.0.0.1:8788/
```

## 说明

本系统用于内部前置筛选和看铺顺序安排，不作为最终签约依据。镇街级人口、经济、消费和餐饮机会中包含 L1/L2 初阶估算，需要用平台截图、租金报价和实地走访继续复核。
